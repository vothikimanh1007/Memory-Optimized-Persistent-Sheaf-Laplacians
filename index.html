Memory-Optimized Persistent Sheaf Laplacians for Real-Time Biomedical Signal Anomaly Detection on Resource-Scarce Platforms

CECIIS 2026 • Interactive Hardware Profiling Demo

Abstract & Contributions

Raw-data streaming in edge-intelligence systems faces a severe computational trilemma involving precision, latency, and energy consumption. This project proposes a hardware-software co-design framework utilizing Persistent Sheaf Laplacians to shift the paradigm from raw data transmission to highly localized, geometry-aware feature extraction on microcontrollers.

  • Algorithmic Compression: 1D symmetric distance matrix flattening reduces topological graph construction memory by 50%.
  • Computational Acceleration: Early-Terminated Union-Find algorithm halts reduction early, achieving sub-8ms latencies.
  • Clinical Validation: Tested on MIT-BIH and PPG-DaLiA datasets, isolating physiological anomalies from motion artifacts.
